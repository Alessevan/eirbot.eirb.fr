# Eirbot's new website

## Development

Serve the website in dev mode with the draft pages

```sh
make dev
```

## Test the production 

Serve the website in production mode without the draft pages

```sh
make test-prod
```

## Build the website

Build in production mode the website in the `public` dir

```sh
make build
```
